# SIGEPED - Frontend (Angular + Bootstrap)

> Interfaz de usuario para el sistema de gestión de pedidos de ESIAD. Desarrollado con Angular y estilizado con Bootstrap.

## 🎯 Objetivo
- Dashboard interno para operarios/administradores
- Simulador de chat de WhatsApp (vista cliente)
- Seguimiento visual de pedidos por estado
- Integración con backend vía servicios HTTP

## 🛠️ Tecnologías
- Angular (v17+ recomendado)
- Bootstrap 5 (CSS Framework)
- TypeScript
- RxJS (para manejo de observables)
- HttpClient (para consumo de API)
